# Random Anime Chatbot

## Requirements
```
virtualenv==20.0.27
pymessenger==0.0.7.0
```

## Prerequisite
``` bash
# create virtual environment
$ virtualenv venv -p python3
# activate virtual environment
$ . venv/bin/activate
# install requirements
(venv) $ pip install -r requirements.txt
```
